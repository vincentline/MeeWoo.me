## 🛠️ 工具分类

### 文件操作
- Read - 读取文件内容
- Edit - 修改文件内容
- Write - 创建或覆盖文件
- DeleteFile - 删除文件或目录
- Glob - 文件模式匹配
- LS - 列出目录内容

### 命令执行
- RunCommand - 执行终端命令
- CheckCommandStatus - 检查命令状态
- StopCommand - 终止运行中命令

### 技能调用
- Skill - 调用项目技能

### 其他工具
- Grep - 搜索文件内容
- TodoWrite - 管理任务列表
- WebSearch - 搜索互联网
